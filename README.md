# MAC address command-line utility

## Run this utility using command line:

Step 1: Execute below command from project root directory:


`$ mvn clean install`


Step 2: Run Jar file using below command:

`$ java -jar target/macaddress.jar`


Example:

`$ java -jar target/macaddress.jar`
